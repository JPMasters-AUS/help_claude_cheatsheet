# Contributing

Thanks for wanting to improve this cheat sheet!

## How to Suggest Changes

1. **Open an Issue** — describe what's missing, wrong, or confusing
2. **Or submit a Pull Request:**
   - Fork this repo
   - Create a branch (`git checkout -b fix/my-improvement`)
   - Make your changes to `help_claude`
   - Test it locally: `bash help_claude` and check each section
   - Commit with a clear message
   - Open a PR

## Guidelines

- Keep language simple and beginner-friendly
- Test your changes by running the script before submitting
- One change per PR where possible
