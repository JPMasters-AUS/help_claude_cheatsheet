# help_claude — Claude Code Terminal Cheat Sheet

A quick-reference command for Claude Code beginners. Type `help_claude` in any terminal to see commands, shortcuts, and tips.

## Install

**Paste this single line into your terminal:**

```bash
curl -fsSL https://raw.githubusercontent.com/JPMasters-AUS/help_claude_cheatsheet/main/install.sh | bash
```

That's it. Open a new terminal tab and type `help_claude` to confirm it worked.

**Alternative — clone and install manually:**

```bash
git clone https://github.com/JPMasters-AUS/help_claude_cheatsheet.git
cd help_claude_cheatsheet
bash install.sh
```

## Usage

```
help_claude              # Full cheat sheet
help_claude start        # How to launch Claude Code
help_claude resume       # How to resume previous sessions
help_claude slash        # Slash commands (/help, /clear, etc.)
help_claude keys         # Keyboard shortcuts
help_claude permissions  # Permission modes explained
help_claude tips         # Beginner tips
```

## Uninstall

```bash
curl -fsSL https://raw.githubusercontent.com/JPMasters-AUS/help_claude_cheatsheet/main/uninstall.sh | bash
```
